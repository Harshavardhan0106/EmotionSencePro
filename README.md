## Overview
EmotionSencePro is a machine learning project that detects human emotions using the FER2013 dataset and Logistic Regression model.

## Features
- Emotion classification
- Data preprocessing
- Model evaluation
- Flask API

## Dataset
FER2013 dataset is not included due to size limits.
Download: https://www.kaggle.com/datasets/msambare/fer2013

## How to Run
pip install -r requirements.txt
python app.py
